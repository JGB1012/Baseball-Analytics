# MLB First Basemen Analysis in Python

This project analyzes MLB first basemen using Python and common baseball performance metrics. The goal is to compare players across offensive production and value metrics, then create a simple composite score to rank overall performance.

## Tools Used
- Python
- pandas
- matplotlib
- numpy

## Metrics Included
- AVG
- OBP
- SLG
- OPS
- HR
- RBI
- WAR

## Project Goals
- Clean and analyze player-level data
- Compare first basemen across key offensive metrics
- Create visualizations for performance analysis
- Build a simple weighted ranking model

## Outputs
- Ranked CSV of first basemen
- OPS bar chart
- HR vs WAR scatter plot
- Overall score ranking chart

## How to Run
```bash
pip install -r requirements.txt
python src/analysis.py
